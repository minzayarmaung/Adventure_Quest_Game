# 📖 About

**Legends of the Shattered Realm** is a handcrafted **2D Pixel-Art Adventure RPG** built entirely with **Pure Java Swing**, without relying on commercial game engines like Unity or Godot.

Step into a mysterious world filled with forgotten ruins, hidden secrets, and meaningful choices. Explore beautiful handcrafted environments, interact with unique characters, solve puzzles, and uncover the truth behind the shattered realm.

Designed as both a passion project and a technical showcase, this game demonstrates object-oriented programming, game architecture, AI behaviors, rendering systems, animation, and storytelling using nothing but Java.

> 🎨 **Retro Pixel Art** • 🧠 **Critical Thinking** • 🗺️ **Adventure** • 🎵 **Immersive Music** • 💬 **Story-Driven** • ☕ **Pure Java**

# 🧠 Critical Thinking Over Combat

Not every challenge can be overcome with brute force.

In **Legends of the Shattered Realm**, every obstacle is a puzzle waiting to be solved.

Observe your surroundings, gather clues, explore different choices, and think carefully before making a decision. Your greatest weapon is your ability to reason, adapt, and uncover the right solution.

> **Every choice matters. Every decision shapes your journey. Intelligence—not force—is the true key to survival.**

# 🎮 Controls

The game is played entirely with your mouse.

| Mouse | Action |
|--------|--------|
| 🖱️ Left Click | Interact with characters, objects, and confirm selections. |
| 🖱️ Right Click | Open the interaction menu and choose from available actions. |

# 🚀 Getting Started

## Requirements

- Java 17 or later
- IntelliJ IDEA (Recommended)

## Clone the Repository

```bash
git clone https://github.com/minzayarmaung/Adventure_Quest_Game.git
```

Open the project using your preferred Java IDE and run the main class.
